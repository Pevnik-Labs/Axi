# T-Axi Demo

In this directory you can find the demo of T-Axi, i.e. a bunch of plain text files (file extension `.axi.demo`) which propose how the language will look and feel from the user's point of view.

You should read the files in the following order:
1. [Theoretical Intro](Layers.md)
1. [Intro](Tutorial/Intro.axi.demo)
1. [Propositional logic](Tutorial/PropositionalLogic.axi.demo)
1. [Classical propositional logic](Tutorial/ClassicalPropositionalLogic.axi.demo)
1. [First-order logic](Tutorial/FirstOrderLogic.axi.demo)
1. [Chaining](Tutorial/Chaining.axi.demo)
1. [Type quantification](Tutorial/TypeQuantification.axi.demo)
1. [Higher-order logic](Tutorial/HigherOrderLogic.axi.demo)
1. [Programming](Tutorial/Programming.axi.demo)
1. [Classical higher-order logic](Tutorial/ClassicalHigherOrderLogic.axi.demo)
